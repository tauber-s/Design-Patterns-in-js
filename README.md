# Design Patterns in JavaScript (refactoring.guru)

This repository aims to study, implement, and document **all Design Patterns presented on [[refactoring.guru](https://refactoring.guru/)]**, using **JavaScript** with clear, simple, and well-structured examples.

The project is focused on **learning, practice, and future reference**.

---

## Study Resources

- https://refactoring.guru/design-patterns
- Book: *Dive Into Design Patterns* – Alexander Shvets

---

## Project Goals

- Implement **all Creational, Structural, and Behavioral patterns**
- Follow the **structure and terminology** used by refactoring.guru
- Keep examples **simple, isolated, and well-commented**
- Use **modern JavaScript (ES Modules)**
- Make it easy to compare similar patterns
